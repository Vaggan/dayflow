# DayFlow

> A calm family assistant that helps everyone understand today.

---

## Why DayFlow exists

Modern family life is fragmented across many tools:
calendars, school systems, weather apps, reminders, and notes.

DayFlow exists to bring this information together into one shared view of the day,
so the family can move through daily life with more calm and clarity.

---

## Core idea

DayFlow is not a new system to manage.

It is a way to **see what already exists**.

It aggregates information from existing services and presents it in a simple, shared perspective of the current day.

---

## Principles (early draft)

- Calm over complexity  
- Shared first  
- Integrate, don't replace  
- Product before technology

## What this is NOT

DayFlow is not:
- a new calendar
- a task manager
- a social platform
- a news feed
- a smart home controller
